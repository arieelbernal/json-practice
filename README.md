# json-practice

## Rutas

### Usuarios (`/usuarios`)

- GET `/usuarios`
- GET `/usuarios/:id`
- POST `/usuarios`
- POST `/usuarios/login`
- PUT `/usuarios/:id`
- DELETE `/usuarios/:id`

### Productos (`/productos`)

- GET `/productos`
- GET `/productos/:id`
- POST `/productos`
- POST `/productos/search`
- PUT `/productos/:id`

### Ventas (`/ventas`)

- GET `/ventas`
- GET `/ventas/:id`
- POST `/ventas`
- POST `/ventas/by-user`
- PUT `/ventas/:id`
- DELETE `/ventas/:id`
